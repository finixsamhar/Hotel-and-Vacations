Hotel and Vacations

An Android application for managing vacations, excursions, and travel details.

Key Features:

  • Secure login (Admin/12345)
  • Add/update/delete vacations
  • Vacation tracking with start/end dates
  • Add/update/delete excursions
  • Excursion management within vacations
  • Real-time database persistence using Room
  • Real-time searching feature
  • Vacations report feature
